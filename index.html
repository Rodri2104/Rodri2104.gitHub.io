¿Qué es una página web?

Se conoce como página Web, página electrónica o página digital a un documento digital de carácter multimediático (es decir, capaz de incluir audio, video, texto y sus combinaciones), adaptado a los estándares de la World Wide Web (WWW) y a la que se puede acceder a través de un navegador Web y una conexión activa a Internet. Se trata del formato básico de contenidos en la red.

¿Que contenidos muestra mi página web?

Esta página web esta creada por Alejandro Rodríguez Leal y muestra todos los contenidos y tareas dados en la asignatura de Lenguajes de Marca. Consta de un menú con varios apartados con los correspondientes ejercicios realizados en clase. Además incluye un apartado de contacto donde estan todos mis datos personales y redes sociales. Esta muestra el uso de HTML, CSS y JavaScript.
