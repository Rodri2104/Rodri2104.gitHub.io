Uso de Etiquetas de Formato

Etiquetas destinadas a dar sentido a una parte concreta de la página web:


Para destacar una parte del texto se emplea strong.

Para crear un salto de línea utilizamos la etiqueta br.

Para enfatizar un texto utilizamos em.
Habitualmente se representa en cursiva con la etiqueta i

El elemento i representa un tramo de texto desplazado de su contenido circundante sin transmitir ningún énfasis o importancia adicional, y para el cual la presentación tipográfica convencional es texto en cursiva; por ejemplo, una designación taxonómica, un término técnico, una frase idiomática de otro idioma, un pensamiento o el nombre de un barco.


El atributo de título en el elemento abbr representa la expansión de la abreviatura o el acrónimo.

Utilizamos dnf para definiciones

Code para bloques de código informático
Samp para ejemplos

Kdb para texto de teclado

Var para definir una variable
Cite para una cita
